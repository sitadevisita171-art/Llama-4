# Llama-4
  This project integrates AdMob ads (banner &amp; interstitial) with a HTML5 video player. It includes meta verification and responsive UI.   
# Llama AI 😈 Web App

## Features
- Text Q&A
- Image generation
- Voice reply

## Tech Stack
- Frontend: HTML, CSS, JavaScript (React.js)
- Backend: Flask (Python)
- AI: Llama AI (text, image, voice)

## Setup
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `python app.py`
4. Open `http://localhost:5000` in your browser

## Deploy
- Deploy on Glitch or Heroku
- Make sure to set environment variables for API keys
